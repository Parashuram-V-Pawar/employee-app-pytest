# employee-app-pytest
A Python application to manage employee data, built with object-oriented principles and tested using PyTest.

## Features
1. Adding employees
2. Viewing employees
3. Calculating net salary of employees
4. Deleting employee records

## Tech stack
```
-- Language: Python
-- Tools: VSCode, Git
-- Testing: Pytest
```

```

|
|-- test

|

|-- requirements.txt
|-- .gitignore
```

## Installation
```
Clone the repository
-> git clone https://github.com/Parashuram-V-Pawar/employee-app-pytest.git

Move to project folder
-> cd employee-app-pytest

Install dependencies
-> pip install -r requirements.txt

Run the application
-> python main.py
```
